# A set of example dummy alignment genes.

Dummy genes, similar to those generated with
[`make_alignment_dummies()`](https://wilkox.org/gggenes/dev/reference/make_alignment_dummies.md),
for example purposes only.

## Usage

``` r
example_dummies
```

## Format

A data frame with eight rows and three variables:

- molecule:

  the genome

- start:

  the start position of the dummy

- end:

  the end position of the dummy

- gene:

  the name of the dummy gene
